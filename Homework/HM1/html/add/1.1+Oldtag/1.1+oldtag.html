Тег <dl> HTML список определений

ОПИСАНИЕ

Тег <dl > входит в тройку элементов <dl >, <dt >, <dd >, предназначенных для создания списка определений. Каждый такой список начинается с
контейнера <dl >, куда входит тег <dt > создающий термин и тег <dd > задающий определение этого термина.

СИНТАКСИС

элементы dl и dt

ПРИМЕР

Тег <dl > HTML используется для создания списка определений. Тег <dl > контейнер списка определений.
Список созданный с помощью тега <dl >, содержит пары "термин – описание".
Термин в списке определений обозначается тегом <dt > его описание – тегом <dd >
